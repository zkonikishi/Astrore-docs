Astrore

跨平台 Minecraft Java 服务端管理器

基于 Tauri 2 + Rust 构建，支持桌面端本地管理和浏览器远程控制。
内置 AI 运维助手，集成 35+ 模型厂商。

核心特性

🖥️

跨平台桌面端

Windows / macOS / Linux 原生支持，通过 Tauri 2 直接管理本机服务端进程。

🌐

浏览器远程管理

运行 astrore-agent 后通过浏览器管理远程服务器，支持 HTTP API 和 WebSocket 实时推送。

🤖

AI 运维助手

内置 AI 对话，支持 35+ 模型厂商（OpenAI、Claude、Gemini、DeepSeek、通义千问、豆包等），可结合服务器日志分析问题。

📊

实时性能监控

CPU、内存、TPS、MSPT、在线玩家、磁盘空间等指标实时展示，支持历史曲线。

📦

一键下载服务端

内置 Paper、Purpur、Fabric、Forge、Vanilla 等核心下载，支持 Java 运行时自动下载。

🔐

权限与玩家管理

直接编辑 ops.json / whitelist.json / banned-players.json，支持可视化权限组配置。

Agent 网页端

无需桌面环境，通过浏览器管理 Minecraft 服务端。

1

下载 Agent 一体包

从 GitHub Releases 下载 Astrore-Agent-windows.zip 或对应平台包。

2

解压并启动

Windows 双击 start-agent.bat，Linux/macOS 运行 ./start-agent.sh。

3

浏览器访问

打开 http://127.0.0.1:1421 进入网页控制台。

# 手动启动 Agent（自定义端口和令牌）
ASTRORE_BIND=0.0.0.0:1421 \
ASTRORE_TOKEN='your-secret-token' \
./astrore-agent

Agent 同时提供 HTTP API、WebSocket 实时推送和 Minecraft 服务端管理能力。公网部署建议通过 Caddy / Nginx 反向代理启用 HTTPS。

使用指南

🚀 快速开始

  1. 下载并启动 Astrore
  2. 在「启动设置」中选择工作目录和服务端核心
  3. 配置内存和 JVM 参数
  4. 点击「启动服务端」
  5. 在控制台查看实时日志

⚙️ 配置服务端

  1. 进入「配置文件」编辑 server.properties
  2. 调整游戏模式、难度、端口等
  3. 在「插件管理」中安装插件/模组
  4. 在「权限管理」中设置 OP 和白名单

🤖 AI 运维助手

  1. 进入「AI 助手」页面
  2. 点击设置，选择模型厂商（支持 35+ 厂商）
  3. 填入 API Key
  4. AI 会结合当前服务器状态和日志分析问题

🔄 崩溃自动重启

  1. 进入「实例设置」→「崩溃重启」
  2. 启用自动重启
  3. 设置最大重启次数和延迟
  4. 服务端异常退出时自动恢复
